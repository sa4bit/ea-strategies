This EA uses the EMA indicator across 3 selectable timeframes (HTF, ITF, ETF) to determine trade direction. The EMA values are configured per timeframe, and a trade is opened only when the price aligns across the selected EMA filters and ADX filters. You can independently enable any timeframe as a filter. These timeframes act as directional validators. For example, if EMA is enabled on H4 and H1, the price must be above EMA9 on both for a Buy signal to be valid. The actual trade entry is triggered based on the selected Entry Timeframe (e.g., M15), and the Exit Timeframe determines when a trade is closed.
