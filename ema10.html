üìò EMA Strategy Description

This strategy uses the EMA (Exponential Moving Average) across up to three configurable timeframes (HTF, ITF, ETF) to identify directional bias and manage trades with precision.

You can independently enable any timeframe as a filter. These timeframes act as directional validators. For example, if EMA is enabled on H4 and H1, the price must be above EMA9 on both for a Buy signal to be valid.

The actual trade entry is triggered based on the selected Entry Timeframe (e.g., M15), and the Exit Timeframe determines when a trade is closed.

‚úÖ Example Behavior:

  • HTF = H4, ITF = H1, ETF = M15
  • Entry Timeframe = M15
  • Exit Timeframe = M15

‚û§ A Buy trade opens when price closes above EMA9 on M15, only if price is already above EMA9 on H1 and H4.
‚û§ The trade closes when price closes below EMA9 on M15.

If Exit Timeframe is set to H4, the trade will not close on M15 signal but instead wait for H4 condition to break.

üìà ADX Trend Strength Filter (Optional)

You may enable ADX on any timeframe. If enabled, the EA verifies:

  • ADX value > 25, and
  • +DI > -DI (for Buy) or -DI > +DI (for Sell)

Just like EMA, the ADX filter strengthens directional confirmation by filtering out weak or sideways markets. If ADX is enabled on HTF (e.g., H4), it works as a trend-strength filter on that timeframe.

üîÅ Manual Trade Handling

If a trade is manually closed, the EA will not immediately re-open the trade under the same signal. It will wait for a new valid signal to avoid noise and false repetition.

EMA Strategy EA

üìò Strategy Description

This EA uses the EMA indicator across 3 selectable timeframes (HTF, ITF, ETF) to determine trade direction. The EMA values are configured per timeframe, and a trade is opened only when the price aligns across the selected EMA filters.

Entry Timeframe (Dropdown): HTF / ITF / ETF

Exit Timeframe (Dropdown): HTF / ITF / ETF

‚úÖ Entry Conditions:

  • Buy: Price closes above EMA on selected Entry Timeframe
  • Sell: Price closes below EMA on selected Entry Timeframe

‚úÖ Optional Filter:

  • ADX Filter: If enabled, requires ADX > 25 for trend strength confirmation on the Entry Timeframe
