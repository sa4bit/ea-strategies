MACD Strategy EA

üìò Strategy Description

This EA uses the EMA indicator across 3 selectable timeframes (HTF, ITF, ETF) to determine trade direction. The EMA values are configured per timeframe, and a trade is opened only when the price aligns across the selected EMA filters and ADX filters. You can independently enable any timeframe as a filter. These timeframes act as directional validators.

For example, if EMA is enabled on H4 and H1, the price must be above EMA9 on both for a Buy signal to be valid. The actual trade entry is triggered based on the selected Entry Timeframe (e.g., M15), and the Exit Timeframe determines when a trade is closed.

‚úÖ Entry and Exit Behavior:

Entry Timeframe: HTF / ITF / ETF (Selectable)

Exit Timeframe: HTF / ITF / ETF (Selectable)

‚úÖ Entry Conditions:

  • Buy: Price closes above EMA on selected Entry Timeframe and (if selected) price is already above EMA on H1 and H4. The trade closes when price closes below EMA on selected Exit Timeframe (e.g. M15).
  • Sell: Price closes below EMA on selected Entry Timeframe and (if selected) price is already below EMA on H1 and H4. The trade closes when price closes above EMA on selected Exit Timeframe (e.g. M15).

If Exit Timeframe is set to H4, the trade will not close on M15 signal, but instead wait for H4 condition to break.

‚úÖ Optional Filter:

  • ADX Filter: You may enable ADX for trend strength confirmation on any timeframe. If enabled, the EA verifies:

    • ADX value > 25, and
    • +DI > -DI (for Buy) or -DI > +DI (for Sell)

    Just like EMA, the ADX filter strengthens directional confirmation by filtering out weak or sideways markets. If ADX is enabled on HTF (e.g., H4), it works as a trend-strength filter on that timeframe.

    üîÅ Manual Trade Handling

    If a trade is manually closed, the EA will not immediately re-open the trade under the same signal. It will wait for a new valid signal to avoid noise and false repetition.
